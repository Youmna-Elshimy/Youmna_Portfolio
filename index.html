This ongoing project aims to optimize performance in the Bear Hunt event from the mobile game Whiteout Survival. Since the game does not offer a public API, I am currently in the data collection stage, manually logging rallies, troop deployments, and milestone progress. The project focuses on identifying which rally types offer the best reward-to-effort ratio, developing a strategy to assign troops across seven deployments for maximum damage, and predicting when all milestones will be achieved. I also plan to build a performance dashboard to visualize damage by deployment, rally effectiveness, and milestone progress over time. Additionally, I will explore damage forecasting using time series or regression models to support more strategic decision-making during the event.
