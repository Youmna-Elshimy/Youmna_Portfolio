Projects

Assessing Generalisation Gaps in Image Classification

A big data analysis project comparing model performance across ImageNet and ImageNetV2. As project manager of a three-member team, I coordinated roles, set deadlines, and facilitated meetings. This was my first time working with such a large dataset. The code took a significant amount of time to run due to its size, which taught me the importance of optimizing for scale. I handled hyperparameter tuning, trained, tested, and evaluated the tuned model, created visualizations to compare performance before and after tuning, and conducted KS hypothesis testing. I also documented and analysed the results.
Tools used: Python, TensorFlow, Keras, Keras Tuner, scikit-learn, SHAP, pandas, NumPy, matplotlib, seaborn, SciPy, Google Colab, Google Drive
